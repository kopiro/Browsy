This project is about a web browser for classic Mac OS (System 6/7), targeting 68k Macs.

For documentation around the more general "Macintosh SE to the Internet" project, read and update from `~/Projects/MacintoshSE/ALL.md`

Do not run builds unless the user explicitly asks for one. `make docker-build`
is allowed when explicitly requested for verification. Do not run
`make docker-dsk` unless the user explicitly asks for that disk-image build.
